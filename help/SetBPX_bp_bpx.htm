SetBPX[,bp,bpx]
Sets an INT3 (SHORT/LONG) or UD2 breakpoint and optionally assigns a name to it.

arguments
  arg1: Address to put a breakpoint on. This can be an API name [not implemented].

[arg2]: Name of the breakpoint, use quotation marks to include spaces.

[arg3]: Breakpoint type. Can be one of the following options in random order: "ss" (single shot breakpoint), "long" (CD03) and "ud2" (0F0B). You can combine the "ss" option with either "long" or "ud2".

result
 This command sets no result variables.
